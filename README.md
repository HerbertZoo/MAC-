# Mac 一键输入今日日期

这是一个基于 macOS「快捷指令」制作的小工具，可以快速生成当天日期，并自动复制到剪贴板，方便在 Finder 中重命名文件夹或文件。

适合用于论文材料整理、Zotero 文献管理、项目文件夹归档、每日学习笔记命名等场景。

## 为什么做这个快捷指令？

最近在整理 Zotero 文献、论文材料和项目文件夹时，我发现一个很常见但很麻烦的问题：

每次给文件夹重命名，都需要手动输入当天日期，例如：

```text
20260705_文献整理
```

虽然只是几个数字，但每天重复输入不仅麻烦，还容易输错。

所以我用 Mac 自带的「快捷指令」做了一个小工具：一键生成今日日期，并复制到剪贴板。

我的默认命名格式是：

```text
yyyyMMdd_
```

例如今天会自动生成：

```text
20260705_
```

后面可以直接接文件夹主题，例如：

```text
20260705_Zotero文献整理
20260705_论文材料归档
20260705_项目资料整理
```

这样文件夹会按照时间自动排序，后续查找和归档都更加清晰。

## 创建步骤

### 第一步：新建快捷指令

打开 macOS 自带的「快捷指令」App，点击右上角的 `+`，新建一个快捷指令。

### 第二步：设置日期自动复制

依次添加以下 3 个动作：

```text
当前日期
↓
格式化日期
↓
拷贝至剪贴板
```

其中，「格式化日期」的设置如下：

```text
日期格式：自定
格式化字符串：yyyyMMdd_
区域：默认
```

其中，`yyyyMMdd_` 表示输出格式为：

```text
年年年年月月日日_
```

例如：

```text
20260705_
```

如果不想在日期后面添加下划线，也可以改成：

```text
yyyyMMdd
```

### 第三步：设置快捷键并使用

点击快捷指令右上角的 `i` 详情按钮，设置一个自己常用的快捷键。

推荐使用：

```text
Control + Option + D
```

不建议使用 `Command + C`、`Command + V`、`Command + D` 这类快捷键，因为容易和系统或常用软件冲突。

## 使用方法

在 Finder 中重命名文件夹时：

```text
选中文件夹
↓
按 Enter 进入重命名状态
↓
按快捷键，例如 Control + Option + D
↓
按 Command + V 粘贴日期
```

最终效果示例：

```text
20260705_文献整理
```

## 可选日期格式

可以根据自己的文件管理习惯修改日期格式：

```text
yyyyMMdd_        → 20260705_
yyyy-MM-dd_      → 2026-07-05_
yyyy.MM.dd_      → 2026.07.05_
yyyyMMdd-HHmm_   → 20260705-2130_
```

## 推荐命名规范

我个人比较推荐使用：

```text
日期_主题_内容
```

例如：

```text
20260705_Zotero_文献整理
20260705_供应链数字化_机制检验
20260705_开题报告_导师修改意见
20260705_年报案例_新华制药
```

这种命名方式的好处是：

- 按时间自动排序
- 方便后续检索
- 适合长期项目归档
- 不容易出现文件夹命名混乱的问题

## 注意事项

这个快捷指令的功能是「复制今日日期到剪贴板」，不是直接自动输入日期。

所以按下快捷键后，还需要再按一次：

```text
Command + V
```

如果想实现按一次快捷键就自动粘贴，可以进一步添加 AppleScript 动作，但通常需要额外开启辅助功能权限。为了稳定和简单，推荐先使用「复制到剪贴板 + 手动粘贴」的方式。
